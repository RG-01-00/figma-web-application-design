# 🎙️ OSCA Waves - Tech Podcast Platf
**OSCA Waves** is a podcast platform dedicated to exploring the world of computer science through tech talks, AI discussions, programming insights, and more. Hosted by the Open Source Community Annaba (OSCA), this platform brings together tech enthusiasts, developers, and learners in an engaging audio experience.

## 🎯 About The Project

OSCA Waves is more than just a podcast platform - it's a community-driven initiative by **Open Source Community Annaba (OSCA)**. Our mission is to empower individuals and organizations with cutting-edge technology solutions through engaging audio content.

### Why OSCA Waves?
- Learn from experienced tech professionals
- Connect with a thriving open-source community
- Access curated content on computer science topics
-  Stay updated with the latest in tech

## ✨ Key Features

### Core Features
- **Podcast Streaming** - Listen to tech talks, AI discussions, and programming content
- **Categories System** - Browse content by topics (Data Structures, AI, Programming, etc.)
- **User Library** - Save liked episodes and create playlists
- **Search Functionality** - Find specific podcasts easily
- **Episode Management** - Track played episodes and favorites

### User Features
- **Personalized Experience** - Liked episodes and saved content
- **Category Filtering** - Filter episodes based on categories
- **Playlist Creation** - Create and manage custom playlists
- **Multi-language Support** - Available in 10+ languages
